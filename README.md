## 🔐 보안 기능

### 비밀번호 보호
- **초기 비밀번호**: `journalist2025`
- 로그인 후 설정에서 비밀번호 변경 가능
- 7일간 자동 로그인 유지
- 세션 만료 시 자동 로그아웃

### 선별 공유
- 특정 기사만 읽기 전용으로 공유 가능
- 공유 링크를 가진 사람만 접근 가능
- 공유된 기사는 수정/삭제 불가능
- 전문가 자문이나 포트폴리오 공유 시 유용

## 🎯 프로젝트 목표

- **체계적인 기사 관리**: 날짜별, 주제별, KPI 유형별로 기사를 분류하고 관리
- **자동 KPI 측정**: 태그 기반으로 1면, 면톱, 기획 등 주요 지표를 자동 집계
- **포트폴리오 생성**: 웹페이지 형태로 포트폴리오를 생성하고 PDF로 내보내기
- **AI 활용 기반 마련**: 축적된 기사 데이터를 바탕으로 글쓰기 스타일 학습 및 발제 아이디어 도출

## ✨ 주요 기능

### 🔒 **보안 및 접근 제어**
- **비밀번호 인증**: 로그인 후에만 전체 기능 사용 가능
- **세션 관리**: 7일간 자동 로그인 유지
- **비밀번호 변경**: 설정에서 언제든 변경 가능
- **선별 공유**: 특정 기사만 읽기 전용 링크로 공유

### 1. 📊 KPI 대시보드
- **실시간 통계**: 전체 기사, 1면, 면톱, 기획 기사 수를 한눈에 확인
- **시각화 차트**: 
  - KPI 유형별 통계 (막대 차트)
  - 주제 분야별 분포 (도넛 차트)
  - 월별 기사 작성 추이 (라인 차트)
- **기간별 필터**: 연도별, 월별로 데이터를 필터링하여 분석
- **최근 등록 기사**: 최근 5개 기사를 빠르게 확인

### 2. 📚 기사 아카이브
- **전체 기사 관리**: 날짜별로 정렬된 모든 기사를 카드 형식으로 표시
- **강력한 검색**: 제목, 본문, 키워드로 기사 검색
- **다중 필터**: 
  - 주제 분야별 (조선, 항공, 해운, 방산, 정유, 석유화학, 재계)
  - KPI 유형별 (1면, 앞면, 면톱, 게시판톱, 기획)
  - 정렬 방식 (최신순, 오래된순, 제목순)
- **페이지네이션**: 한 페이지당 12개씩 표시
- **상세 보기**: 기사 클릭 시 전체 내용, 메타데이터, 태그 정보 확인

### 3. 📝 기사 등록 및 편집
- **직관적인 폼**: 제목, 본문, 게재날짜 등 필수 정보 입력
- **태그 시스템**: 
  - 주제 분야 태그 (복수 선택 가능)
  - KPI 유형 태그 (복수 선택 가능)
  - 키워드 태그 (쉼표로 구분)
- **메모 기능**: 각 기사에 대한 추가 메모 저장
- **수정/삭제**: 등록된 기사를 언제든 수정하거나 삭제

### 4. 📂 포트폴리오
- **자동 생성**: 등록된 기사를 바탕으로 포트폴리오 자동 생성
- **통계 요약**: 전체 기사 수, KPI 유형별 통계를 한눈에 표시
- **주요 기사 선별**: 1면과 기획 기사를 자동으로 선별하여 표시
- **PDF 내보내기**: 포트폴리오를 PDF 파일로 다운로드

### 5. 🔗 선별 공유
- **기사별 공유 링크**: 각 기사마다 고유한 공유 URL 생성
- **읽기 전용 모드**: 공유된 기사는 조회만 가능, 수정 불가
- **간편한 복사**: 클릭 한 번으로 링크 복사
- **활용 시나리오**: 
  - 선배 기자나 전문가에게 피드백 요청
  - 채용 담당자에게 특정 기사 전달
  - 출판사와 협상 시 실적 증명

### 6. 💾 데이터 관리 (내보내기/가져오기)
- **🧠 스마트 내보내기** (AI 분석용 최적화!)
  - **CSV 전체** ⭐ (초경량! JSON보다 80% 작음, AI 최적)
  - **CSV 월별 분할** (가장 안전한 방법)
  - **JSON 월별 분할**: 1월, 2월... 각각 별도 파일로 다운로드
  - **JSON 분기별 분할**: Q1(1-3월), Q2(4-6월), Q3, Q4로 분할
  - **JSON 경량 모드**: 제목+태그+키워드만 (본문 제외로 파일 크기 90% 감소!)
  - **복합 모드**: 월별 분할 + 경량 모드 조합
  - **선택적 다운로드**: 원하는 월/분기만 선택 가능
- **전체 데이터 내보내기**: 모든 기사를 JSON 파일로 다운로드
- **필터링 내보내기**: 특정 분야/기간/KPI 유형만 선택하여 내보내기
- **데이터 가져오기**: 백업한 JSON 파일을 업로드하여 복원
- **활용 시나리오**:
  - ✅ CSV로 전체 데이터 분석 (대용량 문제 완전 해결!)
  - ✅ 다른 대화창에서 AI 분석 작업
  - ✅ Excel에서도 열람 가능
  - ✅ 월별로 나눠서 트렌드 분석
  - ✅ 정기적인 데이터 백업

| 필드 | 타입 | 설명 |
|------|------|------|

| keywords | array | 검색용 키워드 |
| notes | text | 추가 메모사항 |
| created_at | datetime | 생성일시 (자동) |
| updated_at | datetime | 수정일시 (자동) |

## 🎨 주요 페이지 및 URI

### 메인 페이지
- **URI**: `/` 또는 `/index.html`
- **기능**: 3개 뷰 전환 (대시보드, 기사 관리, 포트폴리오)

### API 엔드포인트
- **GET** `/tables/articles?limit=1000&sort=publish_date`: 전체 기사 조회
- **GET** `/tables/articles/{id}`: 특정 기사 조회
- **POST** `/tables/articles`: 새 기사 등록
- **PUT** `/tables/articles/{id}`: 기사 수정
- **DELETE** `/tables/articles/{id}`: 기사 삭제

## 💡 사용 방법

### 0. 최초 로그인
1. 페이지 접속 시 로그인 화면이 표시됩니다
2. 초기 비밀번호 `journalist2025` 입력
3. 로그인 후 우측 상단 "설정" 버튼에서 비밀번호 변경 권장

### 1. 기사 등록
1. 우측 상단의 "새 기사 등록" 버튼 클릭
2. 제목, 본문, 게재날짜 입력
3. 주제 분야 태그 선택 (복수 선택 가능)
4. KPI 유형 태그 선택 (복수 선택 가능)
5. 키워드 입력 (쉼표로 구분)
6. 필요시 메모 작성
7. "저장" 버튼 클릭

### 2. 대시보드 활용
1. 상단 네비게이션에서 "대시보드" 클릭
2. 연도/월 필터로 기간 설정
3. KPI 카드에서 주요 지표 확인
4. 차트로 시각화된 통계 분석
5. 최근 등록 기사 빠르게 확인

### 3. 기사 검색 및 관리
1. "기사 관리" 탭 클릭
2. 검색창에 키워드 입력
3. 주제/KPI 필터로 세밀하게 필터링
4. 기사 카드 클릭하여 상세 보기
5. 상세 페이지에서 수정/삭제 가능

### 4. 포트폴리오 생성
1. "포트폴리오" 탭 클릭
2. 자동으로 생성된 포트폴리오 확인
3. "PDF 내보내기" 버튼으로 다운로드

### 5. 기사 선별 공유
1. 공유할 기사를 클릭하여 상세보기
2. "공유" 버튼 클릭
3. 생성된 링크를 복사하여 전달
4. 받은 사람은 로그인 없이 해당 기사만 조회 가능

### 6. 데이터 내보내기 (다른 대화창에서 분석용)
1. 우측 상단 "데이터 관리" 버튼 클릭
2. **🧠 스마트 내보내기** 선택
3. 내보내기 방식 선택:
   - **⭐ CSV 전체 (최고 추천!)**: 모든 기사를 초경량 CSV로 (JSON보다 80% 작음!)
   - **CSV 월별 분할**: 각 월마다 CSV 파일
   - **JSON 월별/분기별**: 필요시 JSON 형식도 가능
4. 연도 선택 (전체 또는 특정 연도)
5. 월/분기 선택 (체크박스)
6. "스마트 내보내기 시작" 클릭
7. CSV 파일 다운로드 → **Excel에서도 확인 가능!**
8. **다른 대화창**에서 CSV 업로드 → AI 분석 성공! ✨

**예시:**
```
✅ "CSV 전체" 선택 → articles_2025.csv (1-2MB, AI 처리 가능!)
✅ "전체 기사 트렌드 분석해줘" → 완벽하게 작동!
✅ "발제 아이디어 7개 추천해줘" → 문제 없음!
✅ "키워드 빈도 분석" → Excel에서도 바로 확인!
```

### 7. 데이터 백업 및 복원
1. "데이터 관리" → "전체 내보내기"로 정기 백업
2. 데이터 손실 시 "데이터 가져오기"로 복원
3. 중복 방지: 같은 ID는 자동으로 건너뜀
